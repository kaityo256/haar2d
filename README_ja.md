# ハールウェーブレットを用いた多重解像度解析のサンプル(2次元版)

## 概要

ハールウェーブレット(Haar Wavelet)を使って二次元画像の多重解像度解析をしてみる。簡単な解説を[Qiita](http://qiita.com/kaityo256/items/ace2bd13aab5638cf9be)に書いたので参照のこと。

## Usage

    $ ruby haar2d.rb
